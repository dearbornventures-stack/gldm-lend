# GLDM Lending Protocol

Micro-lending protocol on Solana for Latin America. No bank, no credit check, instant loans.

**Live:** [gldm.netlify.app](https://gldm.netlify.app)

## Features
- Live crypto prices (SOL, USDC, USDT, PAXG, XAUT, BTC) via CoinGecko API
- Lending calculator — see your earnings before depositing
- Borrowing calculator — know your costs before borrowing
- Bilingual (English / Español)
- Phantom wallet connection
- GLDM holder perks: 15% better rates

## Stack
- Single HTML file (no build step, no framework)
- Netlify hosting
- CoinGecko API (free tier, no key required)
- Solana blockchain (GLDM token)

## Deploy
1. Push to GitHub
2. Connect repo to Netlify
3. Done — auto-deploys on every push
